# 红色社会实践活动网页

这是东南大学交通学院2026年寒假社会实践团--星火寻根队的社会实践活动的宣传网页，主题为 “红色足迹：井冈山·南昌社会实践活动”。

## 项目说明

- 使用HTML、CSS和JavaScript构建
- 响应式设计，适配各种设备
- 部署在Cloudflare Pages上

## 部署说明

1. 将本仓库连接到Cloudflare Pages
2. 自动部署，生成网站

## 实践内容

- 井冈山革命遗址调研
- 南昌八一红色文化学习
- 红色文化宣传创新研究
